## Learning Api Authentication
- Create Project
    - Command :-
    ```
        laravel new 2_Api_Jwt_Authentication
        cd 2_Api_Jwt_Authentication
    ```

- Add Database :-
    <p align="center"><a href="https://raw.githubusercontent.com/dharmilweb/2_Api_Jwt_Authentication/main/public/Api_Auth/Input_1.png" target="_blank"><img src="https://github.com/dharmilweb/2_Api_Jwt_Authentication/blob/main/public/Api_Auth/Input_0.png" width="400" alt="Laravel Logo"></a></p>

    - Change .env file :-
        ```html
            DB_CONNECTION=mysql
            DB_HOST=127.0.0.1
            DB_PORT=3306
            DB_DATABASE=1_Authenticaion
            DB_USERNAME=root
            DB_PASSWORD=
        ```
-  Add Tables in Database...
    - Command :-
        ```
            php artisan migrate
        ```
